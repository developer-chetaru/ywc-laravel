- ✅ View all resources
- ✅ Search resources
- ✅ Filter by status (published/draft)
- ✅ Filter by category
- ✅ Toggle publish/unpublish status
- ✅ Delete resources

### 2. User Session Management (`/mental-health/admin/sessions`)
- ✅ View all user sessions (not just own)

- ✅ Filter by date (today/week/month)

- ✅ Cancel sessions
- ✅ View session statistics and revenue

## Answer to Your Question

**NO, functionality is NOT the same for users and admins:**

### Users Can:
- Access their own dashboard
- Browse therapists
- Book sessions
- Track mood
- Access resources
- Use crisis support

### Admins Can:
- ✅ **Everything users can do** (admins can access all user features)
- ✅ **PLUS** additional admin-only features:
  - Manage therapists (approve/reject/activate)
  - Manage resources (publish/unpublish/delete)
  - View all user sessions
  - Cancel any session
  - View platform analytics
  - View user engagement metrics
  - View revenue statistics
